***********************************************
Nonlinear System Modeling, Analysis, and Design
***********************************************

modeling, analyzing, and designing nonlinear feedback systems,
including support for simulation and optimization.  This chapter
describes the primary functionality available, both in the core
python-control package and in specalized modules and sub-packages.
